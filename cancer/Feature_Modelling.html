Feature Reduction with Variance threshold¶

In [22]:
# Remove genes with near-constant expression
selector = VarianceThreshold(threshold=0.01)
X_train_var = selector.fit_transform(X_train)
# Number of features retained
print("Number of features kept:", X_train_var.shape[1])
X_test_var = selector.transform(X_test)

# Get names of features that passed the threshold
retained_feature_names = X_train.columns[selector.get_support()]

# Convert back to DataFrame with retained feature names
X_train_var_df = pd.DataFrame(X_train_var, columns=retained_feature_names)
X_test_var_df = pd.DataFrame(X_test_var, columns=retained_feature_names)

Number of features kept: 19454

We have pre-filtered genes first using variance threshold before applying RFE. We've kept a threshold of 0.01 meaning it removes features (genes) whose variance is less than 0.01 across all samples. We don't wanna keep the threshold too high cause that might risk removing bilogically relevant genes. After performing variance threshold, 19964 features dropped down to 19454. 510 features were removed which is like 2.5% of the total.

Feature Selection with RFE¶

In [25]:
rfecv = RFECV(
    estimator=rf_model,
    step=50,                      
    cv=StratifiedKFold(3),        
    scoring='accuracy',           
    n_jobs=-1
)
rfecv.fit(X_train_var_df, y_train)

# Get selected features
selected_features = X_train_var_df.columns[rfecv.support_]
print("Optimal number of features:", len(selected_features))

# Retrain on selected features
rf_model.fit(X_train_var_df[selected_features], y_train)
y_pred = rf_model.predict(X_test_var_df[selected_features])

print("Reduced Model Accuracy:", accuracy_score(y_test, y_pred))
print("Reduced Model F1 Score:", f1_score(y_test, y_pred))

Optimal number of features: 1804
Reduced Model Accuracy: 0.9769230769230769
Reduced Model F1 Score: 0.9777777777777777

After performing RFE, despite reducing the feature set by over 90%, the model retained virtually identical performance compared to the full feature set, demonstrating the effectiveness of variance thresholding followed by RFE in isolating the most informative genes.
